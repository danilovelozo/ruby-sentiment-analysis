# Sentiment Analysis

A simple sentiment analysis script write in Ruby

## Getting Started

```
git clone git@github.com:danilovelozo/ruby-sentiment-analysis.git

bundle install

cd ruby-sentiment-analysis

ruby lib/sentiment.rb

```
